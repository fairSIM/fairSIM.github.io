fairSIM, a free, open source ImageJ plugin for SR-SIM reconstructions


February 7, 2016

Introduction

This page provides an overview of all resources available for fairSIM - Free Analysis and Interactive Reconstruction for Structured Illumination Microscopy.
All resources are hosted on our public github repositorys, most are mirrored on our institute website.

Plug-in

  • ImageJ / Fiji plugin: latest jar file, source code
  • Quickstart guide / short user manual: PDF, source

Datasets

All raw data files organized in this repository.

Raw image files:

  • OMX LSEC Actin 525nm
  • OMX LSEC Membrane 680nm
  • OMX 200nm-Tetraspecks 680nm
  • OMX U2OS Actin 525nm
  • OMX U2OS Mitotracker 600nm
  • OMX U2OS Tubulin 525nm
  • SLM-SIM 200nm-Tetraspeck 680nm
  • TIRF-SIM Tubulin 525nm
  • Zeiss Actin 515nm (512x512 px crop)
  • Zeiss Actin 515nm
  • Zeiss Mito 620nm (512x512 px crop)
  • Zeiss Mito 620nm

Parameters, OTFs (can be found here in the repository)

  • Parameter files for test data
  • OMX OTFs, in fairSIMs format

The parameter files and OTFs are not needed to run the reconstruction.
They can be used to via the ”load reconstruction” feature, and to compare approximated to experimentally determined OTFs.
